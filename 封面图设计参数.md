## 三、文字排版

### 主标题
```
字体：思源黑体 / 阿里巴巴普惠体
字号：120-150px
颜色：白色 / 渐变
位置：居中，距顶部 400px
字数：5-10字
效果：可加阴影 / 描边
```

### 副标题
```
字体：思源黑体 / 阿里巴巴普惠体
字号：60-80px
颜色：白色
位置：主标题下方 80px
字数：3-8字
```

### 标签文字
```
字体：思源黑体
字号：40-50px
颜色：白色
位置：底部
格式：#标签
```

## 八、快速制作SOP

### 步骤：
1. 打开Canva/稿定设计
2. 选择「小红书封面」模板
3. 设置尺寸 1242x1660px
4. 添加渐变背景
5. 添加主标题（测试名称）
6. 添加副标题（"测一测"等）
7. 添加图标/emoji
8. 添加标签
9. 导出PNG格式

### 每张预计耗时：5分钟
