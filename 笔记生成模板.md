# 小红书笔记自动生成模板

## 一、爆款标题公式

基于数据验证的标题结构（按优先级排序）：

### 公式1：VS对比型（爆款率最高）
```
模板：你该 + [选项A] + 还是 + [选项B] + ？

示例：
- 你该换工作，还是换活法？（155万阅读）
- 你适合大城市拼还是小城市躺？（22万阅读）
- 你会对哪种类型的人上头？年上or年下（93万阅读）
```

### 公式2：测一测型
```
模板：测一测 + 你的 + [结果] + ？

示例：
- 测一测，你天生适合吃哪碗饭？
- 测一测，你的隐藏人格是什么？
- 测一测，你是哪种泪失禁体质？
```

### 公式3：你是哪种型
```
模板：你是 + 哪种 + [类型] + ？

示例：
- 你是哪种泪失禁体质？
- 你是纯血INFP吗？
- 你的爱情底色是什么？
```

### 公式4：发现型
```
模板：我发现了 + [人群] + 的 + [特征] + ！

示例：
- 我发现了INFP的天赋！
- 我发现了魅力的本质
```

## 二、正文结构模板

### 模板结构：
```
【开头痛点】1-2句
↓
【测试介绍】3-5句
↓
【结果展示】4种类型，每种1句
↓
【互动引导】1句
↓
【标签】5-8个
```

### 正文模板：
```
[开头痛点：1-2句描述目标人群的困扰或状态]

[测试介绍：做了这个测试才发现...]

原来[测试主题]分四种类型：
[emoji1] [类型A] - [一句话描述]
[emoji2] [类型B] - [一句话描述]
[emoji3] [类型C] - [一句话描述]
[emoji4] [类型D] - [一句话描述]

[个人感受：1-2句]

你是哪种类型？
👇 评论区告诉我~

#[标签1] #[标签2] #[标签3] #[标签4] #[标签5]
```

## 三、自动生成规则

### 输入参数：
| 参数 | 说明 | 示例 |
|-----|------|------|
| 测试主题 | 测试的核心内容 | INFP天赋测试 |
| 目标人群 | 测试面向的人群 | INFP、敏感型人格 |
| 痛点描述 | 目标人群的困扰 | 不知道自己的天赋是什么 |
| 四种结果 | 测试的四种类型 | 创意梦想家、深度洞察者... |

### 自动生成标题（优先级排序）：
```
1. 你该[选项A]还是[选项B]？
2. 测一测，你的[结果]是什么？
3. 你是哪种[类型]？
4. 我发现了[人群]的[特征]！
```

### 自动生成正文：
```
[痛点] + [测试介绍] + [四种结果] + [互动引导] + [标签]
```

## 四、封面图模板

### 设计要素：
| 元素 | 规范 |
|-----|------|
| 主标题 | 测试名称，字数5-10字 |
| 副标题 | "测一测"或"你是哪种" |
| 配色 | 粉紫渐变（INFP风格） |
| 图标 | 1个emoji或简单图形 |
| 布局 | 上标题、中图标、下副标题 |

### 配色方案：
```
主色调：#FF6B9D（粉色）→ #C44AFF（紫色）渐变
背景：#FFF0F5（淡粉）或 #E8F4FD（淡蓝）
文字：#333333（深灰）
```
